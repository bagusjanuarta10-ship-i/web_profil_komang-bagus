Maksud dan Tujuan Halaman

Halaman website ini disusun sepenuhnya menggunakan struktur Tag Table HTML. Meskipun tabel sebenarnya diperuntukkan untuk mengolah data, pada era awal perkembangan web, tabel sangat sering diandalkan untuk mengatur letak (layout) komponen halaman seperti header, konten, dan sidebar.

Highlight Tugas Web Development

Silakan gunakan menu navigasi di sebelah kanan untuk mengeksplorasi halaman tugas lainnya. Anda bisa melihat riwayat pendidikan saya yang menggunakan tag Ordered List, serta daftar skill teknis yang menggunakan Unordered List.
